## What's in v1.0

### Honeypot services
- SSH listener (Paramiko): banner, KEX, auth capture, post-auth shell
- HTTP/HTTPS listener (FastAPI): fake /wp-admin, /phpmyadmin, /admin; scanner detection; JA3 fingerprint
- FTP listener (pyftpdlib): full command capture
- RDP listener: PDU + CredSSP NLA capture

### Intelligence pipeline
- MaxMind GeoLite2 geolocation + ASN enrichment (local DB, Redis cache)
- AbuseIPDB reputation enrichment (async, non-blocking)
- Tool signature detection: Hydra, Medusa, Masscan, Nmap, sqlmap, Nikto
- MITRE ATT&CK TTP mapper (7 built-in rules, STIX 2.1 data)
- Threat scoring algorithm (0–100, composite formula)

### Alerting
- Telegram bot notifications
- Email (SMTP) notifications
- Slack webhook notifications
- Duplicate suppression (30-min cooldown per IP)
- Four severity levels: low / medium / high / critical

### Reports
- PDF threat intel report (WeasyPrint + Jinja2 CSS template)
- Self-contained HTML report
- Auto-triggered for sessions with score ≥ 60
- Manual trigger via API

### Dashboard
- Live attack map (Leaflet.js + WebSocket)
- HTMX admin dashboard (sessions list, detail, analytics)
- TTP frequency chart, top IPs, attack timeline
- JWT-authenticated REST API (full OpenAPI spec at /docs)

### Infrastructure
- Docker Compose production stack (9 services)
- Network isolation: honeypot-net vs internal-net
- Container hardening: non-root, read-only rootfs, cap_drop:ALL
- Caddy TLS termination with Let's Encrypt
- Redis Streams event bus (AOF persistence)
- Daily PostgreSQL backups + 30-day rolling retention
- Structured JSON logging (structlog) + Prometheus metrics

## v1.1 Roadmap (Stretch Goals)

- **ML anomaly detection:** Isolation Forest trained on session features (timing, attempt count, command sequences). Anomaly score stored in `ml_anomaly_scores` table. Weekly retraining cron.
- **STIX 2.1 export:** `python-stix2` Bundle generation from attacker profiles + TTP matches. API endpoint: `GET /api/sessions/{id}/stix`.
- **TAXII 2.1 server:** taxii2-server integration for threat intel sharing with compatible platforms.
- **Grafana:** Pre-built Grafana dashboard JSON with panels for: sessions/hour, top countries (choropleth), TTP frequency, threat score distribution, alert timeline.
